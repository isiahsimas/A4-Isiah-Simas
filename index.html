How to Create a Simple Portfolio Website



Instructions

  1. Clone this repo as a starting point
  2. Decide on the projects, skills and other content you wish to include
  3. Within index.js - change values such as the id, names and images to suit your style
  4. Create md pages and add content for the item pages
  5. Change index.html to only have current, items and pages
  6. Change index.js to append images and page divs
  7. Change css to allow images to display and style, borders, margins, fonts and tables
  8. Have fun!
Hint: Try using the Bootstrap framework for a quick and simple UI.
